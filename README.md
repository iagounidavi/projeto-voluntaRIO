VoluntaRIO - Extensão Unidavi
=============================

> _Conectando a academia à comunidade através do impacto social._

O **VoluntaRIO** é uma aplicação móvel desenvolvida como parte da disciplina de **Estudo Dirigido e Extensão** da UNIDAVI. O projeto visa facilitar a gestão e a participação de acadêmicos em projetos sociais em Rio do Sul e no Alto Vale do Itajaí.

Propósito
---------

O aplicativo resolve a dificuldade de conexão entre as demandas da comunidade (geridas pela Coordenação de Extensão) e os alunos que precisam cumprir horas curriculares, promovendo:

*   **Cidadania e Desenvolvimento Regional**
    
*   **Sustentabilidade**
    
*   **Interação Dialógica** entre universidade e sociedade.
    

Funcionalidades
---------------

### Para o Acadêmico (Perfil Aluno)

*   **Mapa de Oportunidades:** Visualização geolocalizada de projetos (usando assets locais).
    
*   **Inscrição Simplificada:** Botão "Quero Participar" com feedback imediato.
    
*   **Gestão de Horas:** Gráfico de progresso (Ex: 15h/60h) e histórico de atividades.
    
*   **Meus Certificados:** Área para download de comprovantes de participação.
    

### Para a Coordenação (Perfil Admin/UNIDAVI)

*   **Cadastro de Demandas:** Inclusão de novos projetos com fotos, datas e eixos temáticos.
    
*   **Dashboard de Gestão:** Acompanhamento de voluntários ativos e relatórios de impacto.
    
*   **Validação:** Controle de presença e efetividade das ações.
    

Tecnologias Utilizadas
----------------------

Este projeto foi desenvolvido focando em performance mobile e código limpo:

*   **Linguagem:** Dart
    
*   **Framework:** Flutter (Mobile)
    
*   **Gerenciamento de Estado:** Provider
    
*   **Mapas:** Integração visual customizada (Assets locais offline).
    
*   **Design:** Material Design 3 com identidade visual da UNIDAVI.
    

Como Rodar o Projeto
--------------------

Siga os passos abaixo para executar o aplicativo em seu ambiente local:

1.  Clone o repositório.
    
2.  Entre na pasta do projeto (cd voluntarios\_app).
    
3.  Instale as dependências (flutter pub get).
    
4.  Execute o aplicativo (flutter run).
